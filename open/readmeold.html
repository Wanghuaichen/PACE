BioLogic ILP System HTML Files

This directory (~htdocs/BioLogic) contains the primary HTML and image files for the BioLogic ILP Server pages.

These pages contain JavaScript and call Perl CGI scripts to access the SQL database and execute the ILP inference engine.

The BioLogic ILP System consists of:

HTML, CGI files, YAP (Prolog executable):

  • ~htdocs/BioLogic/{*.html, *.jpg, *.png, *.bmp }
  • ~htdocs/BioLogicDoc/{*.html, *.jpg, *.png, *.bmp }
  • ~cgi-bin/{*.pl, yap.exe, libgmp.dll }
  • ~cgi-bin/examples/*

Plus the following Software:

  • mySQL (version 1.4 or later)
  • Apache Web Server
  • Perl (version 5.8 or later)

Note: ~cgi-bin/aleph.pl is the Prolog source file for the ILP engine, not a Perl CGI script.
